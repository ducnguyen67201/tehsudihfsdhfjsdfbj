# Business AGENTS

Canonical operating context for non-product business work in TrustLoop.

Only load this file when the task is explicitly about business operations such
as GTM, founder outreach, growth, messaging, fundraising prep, or content.

## Purpose

`business/` is the operating system for learning how TrustLoop finds users and
grows before there is a mature team around those functions.

At the current stage:

- `gtm/` is active
- `growth/` is scaffolded for later
- `content/` is scaffolded for later

## Current GTM Entry Point

Start here for founder outreach work:

- `business/gtm/gtm-docs.md`

That file points to:

- the current wedge
- the GTM read order
- the live outreach tracker

## Source Of Truth Split

- Repo docs in `business/` are the strategy and workflow source of truth
- live trackers and execution artifacts are the operating source of truth for
  status, leads, replies, and notes

## Working Rules

- Default to the narrowest wedge that can be tested this week
- Real founder language beats internal opinions
- Short, repeated loops beat broad strategy work
- Update `business/` after meaningful learning, not just after product changes
- Do not load business context for normal code work
